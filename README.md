# Land Verification & Attestation Using Blockchain (HCI Lab Report)

This repository contains the **Human Computer Interaction (HCI)** lab report for the project:

**Land Verification and Attestation Using Blockchain**

## Included Files

- **HCI_Lab_Report.pdf** — The complete lab report (introduction, objectives, system overview, prototype description, cognitive walkthrough evaluation, usability problems, improvements, advantages, limitations, conclusion, references, and appendix).

## Project Overview

This project focuses on **Human Computer Interaction (HCI)** principles and **usability evaluation** of a blockchain-based government system prototype.



The lab report describes an interactive, Figma-based UI/UX prototype for a blockchain-powered land verification system. It also includes a cognitive walkthrough usability evaluation across key user tasks such as:


- Login
- Searching land records
- Uploading verification documents
- Tracking verification status
